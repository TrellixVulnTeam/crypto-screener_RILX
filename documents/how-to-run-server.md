# How to run server?

## Development Mode:

    Use these commands:
      Open two termnial tab

      In the first tab run:
      ```
        ttsc -w
      ```
      and, in the second tab run:
      ```
        nodemon /dist/server.js
      ```

## Production mode

    Use these commands:
      Open two termnial tab

      In the first tab run:
      ```
        ttsc
      ```
      and, in the second tab run:
      ```
        node /dist/server.js
      ```
